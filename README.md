# AI Educational Bear (Armenian AI Initiative)

This project is an AI-powered educational toy (bear) designed to help children learn through interactive conversations.

## What it does
- Uses LLM-based conversational AI
- Custom prompts for educational interaction
- Focused on Armenian language and cultural context

## Goals
- Improve Armenian AI ecosystem
- Build real-world AI products
- Create educational tools for children

## Current Status
Early development phase — working on prompt design and system architecture.
